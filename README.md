# Домашнее задание 1.1–1.3

Набор автотестов на JUnit 5 и Selenium для проверки сценариев поиска в Яндексе и последующих переходов на внешние страницы.

## Требования
- Java 17
- Maven 3
- Переменная окружения `CHROME_DRIVER` должна содержать путь до совместимого `chromedriver` (исполняемый файл не добавлен в репозиторий).

## Запуск тестов
```
mvn test
```

## Что проверяют сценарии
- **1.1.** Поиск по запросу «Гладиолус» с наличием ссылки на Ozon на первой странице выдачи.
- **1.2.** Поиск «курсы валют», переход к официальным курсам и сравнение значений для Лари и Азербайджанского маната.
- **1.3.** Поиск «таблица википедия», переход на страницу «Википедия:Таблицы», повторный поиск «Таблица» и проверка порядка преподавателей в таблице кафедры программирования.
